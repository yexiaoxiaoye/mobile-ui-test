无预览按钮版本 + 壁纸存储位置说明

✅ 预览按钮已移除

现在美化应用中不再有"预览"按钮，输入URL后会自动实时预览，点击"应用壁纸"直接应用。

💾 壁纸保存位置

存储方式： 浏览器 localStorage
存储键名： 'wallpaper_data'
存储内容： JSON格式，包含当前壁纸URL和历史记录
存储位置： 浏览器本地存储

数据结构：

{
  "currentWallpaper": "当前壁纸的URL",
  "history": [
    {
      "url": "壁纸URL",
      "name": "壁纸名称",
      "timestamp": "使用时间"
    }
  ]
}

查看方法：

  1. 打开浏览器开发者工具 (F12)
  2. 切换到 "Application" 或 "存储" 标签
  3. 在左侧找到 "Local Storage"
  4. 选择当前网站域名
  5. 查找键名为 "wallpaper_data" 的项目
