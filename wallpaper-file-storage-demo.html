🎨 美化应用 - 文件存储演示

跨浏览器、跨设备的壁纸配置保存方案

📊 系统状态

存储模式: localStorage
后端服务: 离线
配置文件位置: 浏览器localStorage
API地址: http://localhost:3001/api

🚀 启动后端服务器

要使用文件存储功能，需要先启动后端服务器：

# 1. 安装依赖
cd src/mobile-ui-test
npm install
# 2. 启动服务器
npm start
# 或者使用开发模式（自动重启）
npm run dev

服务器启动后：

  • 配置文件将保存到: mobile-ui-test/image/wallpaper-config.json
  • 可以在不同浏览器和设备间共享壁纸配置
  • 支持自动回退到localStorage（如果服务器不可用）
